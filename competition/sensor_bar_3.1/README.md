#_SENSORBAR VERSION 3.1_

#HOW TO CALIBRATE
1. Align the sensorbar evenly
2. Press A
3. You can restart the calibration process by resetting the hardware

#STAGE OF CALIBRATION
1. White
2. Start/end zone
3. Dark green
4. Orange
5. River
6. Light green
